[lionDRP]

[Features]
- Discord display
- Discord button
- Server Rules Button
- ID
- Steamname
- Playtime
- number of players
- server icon

[Installation]
1. drag & drop the Script of your server
2. open your Server.cfg / CFG Editor in TxAdmin
3. add "ensure lionDRP" to your cfg
4. edit the config the way you like it
4. enjoy it!
